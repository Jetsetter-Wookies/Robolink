Quick Start Example

Python 
# Basic CoDrone EDU flight example
from codrone_edu.drone import *

drone = Drone()
drone.pair()          # Connect to controller

drone.takeoff()       # Take off (~80cm hover)
drone.hover(2)        # Hover for 2 seconds
drone.go("forward", 30, 2) # Forward at 30% power, 2 sec
drone.turn_left(90)  # Turn left 90°
drone.land()          # Land

drone.close()         # Close connection

Getting Started

Two ways to code your CoDrone EDU in Python: browser-based Python for Robolink or desktop PyCharm.

🌐

Option 1 — Python for Robolink (Browser)

Python for Robolink is a browser-based coding environment — no installation required. Ideal for Chromebooks or devices with limited storage.

🔗 Access it at: codrone.robolink.com/edu/python/ — works in Google Chrome on any device (not tablets/phones).

Key Features

📁 File Explorer

Located on the left side. Create and organize Python files and folders. Supports upload of JPEG, PNG, Python, and text files. Right-click for rename/delete options.

🎨 Color Training Tool

Found on the right side. Collect, train and test HSVL color data from the drone's color sensor. Create custom color datasets for use in programs.

💻 Console

Click the "Console" button on the right. Shows output, prompts for user input, and displays error/warning messages.

📱 Compatibility

Works on Chromebooks, Windows, macOS. Requires Google Chrome (latest). Not supported on iPads, tablets, or phones.

ℹ️ Use Python for Robolink if you're on a Chromebook, don't have space for PyCharm, your IT team hasn't installed PyCharm, or you need a quick setup.
💻

Option 2 — PyCharm (Desktop)

1

Install Python

Download from python.org/downloads/. Click the yellow "Download Python 3.X" button for the latest release. Important: If using a school computer, ensure you have admin credentials.

2

Install PyCharm

Download the free Community edition from jetbrains.com/pycharm/download/. If you prefer a different editor or IDE, you'll need to know how to install packages manually.

3

Install the CoDrone EDU Library

Create a new PyCharm project and add the import statement to your main.py. PyCharm will prompt you to install the library automatically.

Python
from codrone_edu.drone import *
💡 If no main.py was generated, right-click your project name → New → Python File.

Connection

Functions for establishing and closing the connection between your program and the CoDrone EDU controller.

pair() Connect controller to program
+
Description
Connects your controller with the program. You can optionally specify the USB port name.
Syntax
drone.pair()
drone.pair(portname)
Parameters
  • portnamestring — Optional. A string containing the port name or number (e.g., 'COM3').
Returns
None
Example
Python

# drone.pair(port_name='COM3')  # pair with specific port

drone.takeoff()
drone.hover(1)
drone.land()
drone.close()
close() Close connection to program
+
Description
Closes the connection of your controller with the program. Always call this at the end of your program.
Syntax
drone.close()
Parameters
None
Returns
None
Example
Python
from codrone_edu.drone import *

drone = Drone()
drone.pair()

drone.takeoff()
drone.hover(1)
drone.land()

Flight Commands

Core functions for controlling drone flight: takeoff, landing, movement, and precision positioning.

takeoff() Take off to ~80cm and hover
+
Description
Makes the drone take off at an average height of 80 centimeters and hover. The drone always hovers for 1 second to stabilize before the next command. Takeoff height cannot be modified.
Syntax
drone.takeoff()
Parameters
None
Example
Python
from codrone_edu.drone import *
drone = Drone()
drone.pair()
drone.takeoff()
drone.hover(3)
drone.land()
drone.close()
land() Soft landing at current position
+
Description
Makes the drone stop all commands, hover, and make a soft landing. Also resets all flight motion variables to 0. Tip: Add a hover() or time.sleep() between takeoff() and land() to prevent the land command from being skipped.
Syntax
drone.land()
Example
Python
from codrone_edu.drone import *
drone = Drone()
drone.pair()
drone.takeoff()
drone.hover(3) # prevents land() from being skipped
drone.land()
drone.close()
emergency_stop() Immediately cut all motors
+
Description
Immediately stops all commands and motors — the drone drops. Resets all flight motion variables to 0. Add a hover() before this to ensure the command is received.
Syntax
drone.emergency_stop()
Example
Python
from codrone_edu.drone import *
drone = Drone()
drone.pair()
drone.takeoff()
drone.hover(3) # recommended before emergency_stop
drone.emergency_stop()
drone.close()
hover(duration) Hover in place for given seconds
+
Description
Makes the drone hover for a given amount of time. With no parameter, hovers indefinitely until given another command.
Syntax
drone.hover(duration)
Parameters
  • durationinteger — Duration in seconds.
Example
Python
drone.takeoff()
drone.hover(3) # hover 3 seconds
drone.land()
go(direction, power, duration) Beginner-friendly directional movement
+
Description
Moves the drone in a given direction at a given power for a given duration. Great for beginners.
Syntax
drone.go(direction, power=50, duration=1)
Parameters
  • directionstring — "forward", "backward", "right", "left", "up", "down"
  • powerint — Power 0–100. Default: 50.
  • durationfloat — Duration in seconds. Default: 1.
Example
Python
drone.takeoff()
drone.go("forward", 30, 1)  # forward 30% power 1s
drone.go("backward", 30, 1) # backward
drone.go("right", 30, 1)    # right
drone.go("left", 30, 1)     # left
drone.land()
move_forward / move_backward / move_left / move_right Precise distance-based movement
+
Description
Moves the drone a precise distance in the specified direction using the optical flow sensor. For best performance, fly over a well-lit, patterned surface.
Syntax
drone.move_forward(distance, units="cm", speed=1)
drone.move_backward(distance, units="cm", speed=1)
drone.move_left(distance, units="cm", speed=1)
drone.move_right(distance, units="cm", speed=1)
Parameters
  • distancefloat — Distance to travel.
  • unitsstring — "cm", "m", "ft", "in". Default: "cm".
  • speedfloat — Speed in m/s. Default: 1.0. Max: 2.0.
Example
Python
drone.takeoff()
drone.move_forward(distance=50, units="cm", speed=1)
drone.move_right(distance=50, units="cm", speed=1)
drone.move_backward(distance=50, units="cm", speed=1)
drone.move_left(distance=50, units="cm", speed=1)
drone.land()
move_distance(x, y, z, velocity) Move relative to current position on all 3 axes
+
Description
Moves the drone by specified distances along X, Y, and Z axes relative to its current position and heading. Non-zero values on multiple axes move simultaneously.
Syntax
drone.move_distance(positionX, positionY, positionZ, velocity)
Parameters
  • positionXfloat — X distance in meters (forward/backward).
  • positionYfloat — Y distance in meters (left/right).
  • positionZfloat — Z distance in meters (up/down).
  • velocityfloat — Speed in m/s. Default: 1.0. Max: 2.0.
Example
Python
drone.takeoff()
# Move forward 0.5m, left 0.5m, up 0.25m simultaneously at 1m/s
drone.move_distance(0.5, 0.5, 0.25, 1)
# Move back 0.75m at 0.75m/s
drone.move_distance(-0.75, 0, 0, 0.75)
drone.land()
turn() / turn_left() / turn_right() / turn_degree() Rotate the drone
+
Functions
  • turn(power, seconds)Turn by power (-100–100). Negative=right, positive=left.
  • turn_left(degree=90)Turn left by degrees using built-in gyroscope. Default: 90°.
  • turn_right(degree=90)Turn right by degrees using built-in gyroscope. Default: 90°.
  • turn_degree(degree, timeout=3)Turn to absolute heading. Takes shortest rotation path.
Example
Python
drone.takeoff()
drone.turn_left()       # 90° left turn
drone.hover(1)
drone.turn_right(45)   # 45° right turn
drone.hover(1)
drone.turn_degree(180) # face 180° from start
drone.land()
set_trim() / reset_trim() / get_trim() Correct drift with roll/pitch trim
+
Description
Set, reset, or get the roll and pitch trim values to correct drone drift. Trim is saved even after power-off until changed or reset.
Syntax
drone.set_trim(roll, pitch) # -100 to 100
drone.reset_trim() # resets to 0
drone.get_trim() # returns [roll, pitch]
Example
Python
import time
from codrone_edu.drone import *
drone = Drone()
drone.pair()
# Drone drifts right → trim roll left
drone.set_trim(-5, 0)
time.sleep(1) # required before takeoff when setting trim
drone.takeoff()
drone.hover(3)
drone.land()
drone.reset_trim()
drone.close()
avoid_wall() / keep_distance() Sensor-based wall avoidance
+
Functions
  • avoid_wall(timeout=2, distance=70)Fly forward and stop when an object is detected within distance cm. Sensor range: 0–100cm.
  • keep_distance(timeout=2, distance=50)Fly forward until distance cm from object, then maintain that distance. Sensor range: up to 150cm.
Example
Python
drone.takeoff()
drone.avoid_wall(10, 50) # run 10s, stop 50cm from wall
drone.keep_distance(10, 60) # maintain 60cm for 10s
drone.land()

Flight Variables

Low-level control of roll, pitch, yaw, and throttle for precise custom movements.

⚙️ How it works: Set flight variables with set_roll(), set_pitch(), set_yaw(), set_throttle(), then call move(duration) to execute. Variables persist until reset.
set_pitch / set_roll / set_yaw / set_throttle Set flight motion variables (-100 to 100)
+
Functions
  • set_pitch(power)Forward/backward. Positive=forward, negative=backward.
  • set_roll(power)Left/right. Positive=right, negative=left.
  • set_yaw(power)Rotation. Positive=left, negative=right.
  • set_throttle(power)Altitude. Positive=up, negative=down.
Example — Combined movement
Python
drone.takeoff()
drone.set_pitch(50)    # set forward
drone.set_roll(20)     # drift right slightly
drone.move(2)            # execute for 2 seconds
drone.reset_move_values() # reset all to 0
drone.land()
move(duration) Execute current flight variables
+
Description
Executes movement based on the currently-set flight variables. With a duration, runs for that many seconds. With no parameter, moves indefinitely. Must call takeoff() first.
Syntax
drone.move(duration)
reset_move_values() / get_move_values() Reset or read current flight variable state
+
Description
reset_move_values() resets roll, pitch, yaw, and throttle to 0. get_move_values() returns a list: [roll, pitch, yaw, throttle].
Example
Python
drone.set_roll(30)
drone.set_pitch(40)
values = drone.get_move_values()
print("roll:", values[0], "pitch:", values[1])

Sensors — Range

Use the front and bottom range sensors to measure distance and detect objects.

get_front_range(unit="cm") Distance to object in front (0–150cm)
+
Description
Returns distance from the drone to the surface directly in front. Returns 999 when out of range (>1.5m) or timed out. Returns -10 or 0 on sensor error.
Syntax
drone.get_front_range()
drone.get_front_range(unit="cm") # "m", "mm", "in" also available
Example
Python
import time
from codrone_edu.drone import *
drone = Drone()
drone.pair()
start = time.time()
# Grab drone and move it toward/away from a wall
while time.time() - start < 20:
    print(drone.get_front_range())
    time.sleep(0.05)
drone.close()
get_bottom_range(unit="cm") / get_height(unit="cm") Distance from drone to surface below
+
Description
Both functions use the bottom range sensor. Note: When sitting on a surface, the sensor turns off (returns 0) so the color sensor can function. It is safe to treat 0 as "drone is on the ground." Returns 999.9 when out of range.
Example
Python
drone.takeoff()
drone.set_throttle(50)
while drone.get_height("cm") <= 100:
    drone.move()
    print(drone.get_height("cm"))
drone.land()
detect_wall(distance=50) Returns True when object is within threshold
+
Example
Python
drone.takeoff()
if drone.detect_wall(50): # True if object within 50cm
    print("Wall detected!")
else:
    print("No wall detected.")
drone.land()

Sensors — Optical Flow

📍 Note: Position values reset to 0 when the drone takes off or the battery is re-inserted. X = forward/backward, Y = left/right, Z = up/down.
get_pos_x() / get_pos_y() / get_pos_z() Current drone position from takeoff point
+
Syntax
drone.get_pos_x(unit="cm")
drone.get_pos_y(unit="cm")
drone.get_pos_z(unit="cm")
Example — Fly 100cm forward using position tracking
Python
drone.takeoff()
start = drone.get_pos_x("cm")
drone.set_pitch(15)

while drone.get_pos_x("cm") - start < 100:
    drone.move()

drone.hover(1)
drone.land()
get_flow_velocity_x() / get_flow_velocity_y() Raw optical flow velocity data
+
Description
Returns raw data proportional to drone velocity in the X (forward/back) or Y (left/right) direction from the optical flow sensor.
Example
Python
drone.takeoff()
drone.set_pitch(50)
drone.move(1)
print(drone.get_flow_velocity_x())
drone.land()
get_position_data(delay=0.01) Return all position data as a list
+
Returns
List: [time, x, y, z]
Example
Python

Sensors — Gyroscope

Read acceleration and angle data from the built-in gyroscope sensor.

🌀 Note: Angle values reset to 0 when the drone takes off or battery is re-inserted. At rest, the Z-axis acceleration reads ~9.8 m/s² due to gravity.
get_accel_x() / get_accel_y() / get_accel_z() Acceleration on each axis
+
Axes
  • get_accel_x()X acceleration — forward/backward
  • get_accel_y()Y acceleration — left/right
  • get_accel_z()Z acceleration — up/down (reads ~9.8 at rest)
Example
Python
import time
from codrone_edu.drone import *
drone = Drone()
drone.pair()
start = time.time()
# Grab and tilt drone to see acceleration values
while time.time() - start < 20:
    print(drone.get_accel_x(), drone.get_accel_y())
    time.sleep(0.05)
drone.close()
get_angle_x() / get_angle_y() / get_angle_z() Tilt angles in degrees (roll, pitch, yaw)
+
Axes
  • get_angle_x()Roll angle (left/right tilt)
  • get_angle_y()Pitch angle (forward/backward tilt)
  • get_angle_z()Yaw angle (rotation)
Example
Python
while time.time() - start < 20:
    print("Roll:", drone.get_angle_x(),
          "Pitch:", drone.get_angle_y(),
          "Yaw:", drone.get_angle_z())
    time.sleep(0.05)

Sensors — Color

Detect colors using the drone's bottom-facing color sensor. Works with the Programming Mat and custom color training.

🎨 Tip: The color sensor activates when the drone is sitting on a surface (bottom range sensor is 0). Use it with the CoDrone EDU Programming Mat for best results.
get_color_data() Raw HSVL color data from color sensor
+
Returns
A list of HSVL (Hue, Saturation, Value, Lightness) data from the color sensor.
Example
Python
from codrone_edu.drone import *
drone = Drone()
drone.pair()
data = drone.get_color_data()
print("HSVL:", data)
drone.close()
get_color() Detect color name from Programming Mat
+
Description
Returns a string with the detected color name when using the CoDrone EDU Programming Mat. Common return values include "red", "blue", "green", "yellow", "white", "black".
Example
Python
from codrone_edu.drone import *
drone = Drone()
drone.pair()
color = drone.get_color()
print("Detected color:", color)
if color == "red":
    drone.set_drone_LED(255, 0, 0, 100)
drone.close()

Sensor Guide

Overview of all sensors on the CoDrone EDU and how to use them effectively.

📡 Front Range Sensor (ToF)

Range~9mm – 150cm
Functionget_front_range(), detect_wall(), avoid_wall()
NoteCannot read exactly 0 due to light scatter

⬇️ Bottom Range Sensor (ToF)

Range0 – 150cm
Functionget_bottom_range(), get_height()
NoteTurns off when on surface (returns 0) so color sensor can work

👁️ Optical Flow Sensor

TracksX/Y position and velocity
Functionsget_pos_x/y/z(), get_flow_velocity_x/y()
Best surfaceWell-lit, patterned surface for accuracy
ResetValues reset on takeoff or battery reinsertion

🌀 Gyroscope / IMU

Tracks3-axis acceleration and angle
Functionsget_accel_x/y/z(), get_angle_x/y/z()
Z at rest~9.8 m/s² (gravity)

🎨 Color Sensor

ReadsHSVL (Hue, Saturation, Value, Lightness)
Functionsget_color(), get_color_data()
Active whenDrone is sitting on a surface
Best useProgramming Mat or custom color training
🔗 Full sensor guide: docs.robolink.com/docs/CoDroneEDU/Resources/Sensor-Guide
